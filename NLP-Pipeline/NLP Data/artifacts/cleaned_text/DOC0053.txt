Evidence Imaging

Evidence Imaging
•
Forensic evidence has to be imaged and duplicated
•
The image can be used for forensic analysis
•
The original evidence item must not be used for analysis
•
Imaging creates a bit by bit image of the evidence item
•
The evidence item could be a hard drive,USB,SSD
•
All the files and folders have to be hashed
•
FTK imager from AccessData and Autopsy are some of the tools we shall use to create forensic images
of evidence.
